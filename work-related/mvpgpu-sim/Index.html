Release

Vx.x means that this is a new feature release version Vx.x.x means that this a bug fix version which based on the Vx.x version

  • MVPGPU-SIM V1.0 [Download]

    • Support OpenCL API v1.2

    • Support OpenCL CTS

  • MVPGPU-SIM V1.1 [Download]

    • Support Pytorch based on OpenCL

  • MVPGPU-SIM V1.2 [Download]

    • Support Graphics rendering

    • Support Graphics pipeline

  • MVPGPU-SIM V1.3 [Download]

    • Support OpenGL API v3.3

    • Support OpenGL “hello world”

  • MVPGPU-SIM V1.4 [Download]

    • Support OpenGL CTS

  • MVPGPU-SIM V1.5

    • Support MVP-libs

  • MVPGPU-SIM V1.6

    • Performance Tool

    • Pipeline visualization

  • MVPGPU-SIM V1.7

    • Power evaluation

  • MVPGPU-SIM V1.8

    • Tensor Core

  • MVPGPU-SIM V1.9
